📡 Protocol Quick Facts
EDLEmergency Download Mode — USB PID 0x9008
SaharaBoot protocol used before Firehose loads
FirehoseXML-based flash protocol over USB bulk transfer
Loaderprogrammer.mbn / prog_emmc_firehose*.mbn
Sector size512 bytes (eMMC) or 4096 bytes (UFS)
LUNLogical Unit Number — 0 = main flash on most devices
GPTGUID Partition Table — locates partitions by name
udev ruleLinux: copy Drivers/51-edl.rules → /etc/udev/rules.d/
WindowsRequires libusb/UsbDk — run as Administrator
